# Go-Learning

A collection of small Go projects, experiments, and Exercism exercises as I learn Go.

This repository is a personal learning space for practicing Go fundamentals, exploring the Go ecosystem, and building familiarity with backend development concepts.

## Structure

```text
Go-Learning/
│
├── exercism/        # Exercism Go exercises
│
├── projects/        # Small Go projects and experiments
│
└── README.md
```

## Topics Covered

- Go fundamentals
- Structs and data modeling
- Functions and packages
- Interfaces
- Error handling
- Testing
- JSON and file handling
- HTTP requests and APIs
- Database interactions

## Purpose

The goal of this repository is to build practical experience with Go through consistent practice, small projects, and hands-on experimentation.
